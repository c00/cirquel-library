The Crowdsourced Circus Library for Aerial Silks, Hoops, Trapeze, Pole and Acro Balance! Come into the Cirquel!

Cirquel is a free, crowdsourced and open source circus library that aims to create a common dictionary for all things acrobatics. Whatever your cirque preferences are, Aerial Silks, Aerial Hoops / Lyra, Trapeze, Pole Dancing and Acro Balance, we got you covered.

Cirquel contains images and videos, along with crowdsourced information about skill requirements, strength requirements etc.

Find tricks you love, learn them, add your own version of them, come into the Cirquel!

Cirquel is open-source. If you wish to contribute, we can use everything from translators, to designers to developers, and of course content creators! Check out the website or the github repo for more information on contributing. (Link will follow shortly...)
